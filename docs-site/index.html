Kairos
JavaScript Date Library

Revolutionary zero-dependency JavaScript date/time library with modular architecture and dynamic holiday system.

Get Started View Examples 
import kairos from '@oxog/kairos';

const date = kairos('2024-06-15');
const formatted = date.format('YYYY-MM-DD');
const isHoliday = date.isHoliday();

Why Choose Kairos?

Zero Dependencies

No external dependencies means smaller bundle sizes and better performance.

Modular Architecture

Plugin-based system lets you load only the functionality you need.

TypeScript Support

Full TypeScript support with comprehensive type definitions.

Tree Shakeable

Only bundle what you use for optimal performance.

High Performance

Optimized for speed with efficient date calculations.

Immutable Operations

All date operations return new instances for safer code.

Quick Start

Installation

npm install @oxog/kairos

Basic Usage

import kairos from '@oxog/kairos';

const date = kairos('2024-06-15');
console.log(date.format('YYYY-MM-DD'));
// → 2024-06-15

const tomorrow = date.add(1, 'day');
console.log(tomorrow.format('dddd'));
// → Sunday
