Kairos

Main constructor function for creating date instances.

kairos(input?: DateInput, options?: KairosOptions): KairosDate

Parameters

input DateInput

Date input (string, Date, number, array, or object)

options KairosOptions

Configuration options

Examples

// Current date
const now = kairos();

// From string
const date = kairos('2024-06-15');

// From Date object
const fromDate = kairos(new Date());

// From components
const fromComponents = kairos({ year: 2024, month: 5, day: 15 });

KairosDate

Main date instance class with all manipulation and formatting methods.

Methods

format(format: string): string

Format date according to specified format string.

const date = kairos('2024-06-15');
console.log(date.format('YYYY-MM-DD')); // 2024-06-15
console.log(date.format('MMMM Do, YYYY')); // June 15th, 2024

add(amount: number, unit: TimeUnit): KairosDate

Add specified amount of time units to the date.

const date = kairos('2024-06-15');
const tomorrow = date.add(1, 'day');
const nextMonth = date.add(1, 'month');

subtract(amount: number, unit: TimeUnit): KairosDate

Subtract specified amount of time units from the date.

const date = kairos('2024-06-15');
const yesterday = date.subtract(1, 'day');

Formatting

Kairos provides comprehensive date formatting with various tokens:

Token Description Example
YYYY 4-digit year 2024
MM 2-digit month 06
DD 2-digit day 15
MMMM Full month name June
dddd Full day name Saturday
